# Triage Labels

| 技能里的角色 | 本仓库用的标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 还没看过，需要整理 |
| `needs-info` | `needs-info` | 信息不够，等补充 |
| `ready-for-agent` | `ready-for-agent` | 写清楚了，可以让 AI 直接做 |
| `ready-for-human` | `ready-for-human` | 需要人亲自改 |
| `wontfix` | `wontfix` | 不做了 |

任务文件示例：

```markdown
Status: ready-for-agent

# 修复手机端音量条溢出

……正文……
```
